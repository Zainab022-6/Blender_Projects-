📊 Technical Specs
* **Blender Version:** 5.1
* **Render Engine:** Eevee 
* **Style:** Stylized Low-Poly
* **Materials:** Materials and basic procedural shaders are included and packed into the `.blend` file.

---

## 📂 File Description
* `low-poly-well.blend` - The core Blender file containing the 3D model, lighting setup, and materials.
* `low-poly-well.blend1` - The automatic backup file created by Blender (can be ignored or used if the main file gets corrupted).
